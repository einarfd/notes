# Notes

AI-friendly note-taking solution with MCP integration.

## Installation

```bash
uv sync
```

## Usage

Run the MCP server:

```bash
uv run notes
```

## Development

Run tests:

```bash
uv run pytest
```

Run linting:

```bash
uv run ruff check src tests
```

Type checking:

```bash
uv run mypy src
```
